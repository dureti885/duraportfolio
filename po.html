Projects

City-Skyline

this project, I created a visually appealing image of a city skyline using HTML and CSS. The goal was to design a responsive and aesthetically pleasing representation of a cityscape that could adapt to different screen sizes. Key Features: 1. HTML Structure: The project utilizes semantic HTML to create a well-organized layout. I employed

elements to represent different buildings in the skyline, ensuring clarity and maintainability. 2. CSS Styling: I applied CSS for styling, using gradients to create depth and shadows to give a three-dimensional effect to the buildings. Media queries were implemented to ensure the skyline scales appropriately on various devices. 3. Responsive Design: Special attention was given to responsiveness. The skyline adjusts seamlessly, ensuring that the design remains visually striking on both desktop and mobile screens. This project not only showcases my skills in HTML and CSS but also demonstrates my ability to create engaging visual content that enhances user experience.

Coffe Menu

In this project, I developed a coffee menu website that showcases various coffee options in an engaging and user-friendly format. The objective was to create an attractive digital menu that highlights the unique offerings of a coffee shop. Key Features: 1. HTML Structure: The menu is structured using semantic HTML, with sections for different categories like espresso, brewed coffee, and specialty drinks. This organization makes the content easy to navigate. 2. CSS Styling: I utilized CSS for visual appeal, incorporating vibrant colors, appealing fonts, and images of the coffee drinks. Hover effects were added to enhance interactivity, making the menu more engaging for users. 3. JavaScript Functionality: JavaScript was employed to add interactivity, including features like a filter function to sort coffee types and a modal for detailed descriptions of each drink. This enhances user experience by allowing visitors to find their favorite coffee easily. 4. PHP Backend: The project includes a PHP backend to handle dynamic content. This allows for easy updates to the menu without modifying the HTML directly. I implemented features like a contact form and a database connection to manage customer inquiries. 5. Responsive Design: The layout was designed to be fully responsive, ensuring that it looks great on all devices. Media queries were used to adjust design elements based on screen size. This project showcases my ability to integrate HTML, CSS, JavaScript, and PHP, creating a comprehensive and interactive coffee menu that enhances the overall user experience.
